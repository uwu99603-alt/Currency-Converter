## Как получить API-ключ

В данном проекте используется бесплатный API exchangerate-api.com. Для базового использования ключ не требуется, так как мы используем публичный эндпоинт без аутентификации (лимит около 1500 запросов в месяц). Если вы хотите использовать свой ключ, зарегистрируйтесь на сайте exchangerate-api.com, получите ключ и замените URL в функции get_exchange_rate на следующий: https://v6.exchangerate-api.com/v6/ВАШ_КЛЮЧ/latest/{from_currency}

## Установка и запуск

Убедитесь, что у вас установлен Python версии 3.7 или выше. Установите библиотеку requests с помощью команды pip install requests. Скачайте файл currency_converter.py и сохраните его в отдельной папке. Запустите приложение командой python currency_converter.py.

## Примеры использования

Пример 1: конвертация 100 евро в доллары США. Выберите из валюты EUR, в валюту USD, введите сумму 100 и нажмите кнопку "Конвертировать". Результат покажет, например, 106.50 USD в зависимости от текущего курса.

Пример 2: конвертация 5000 рублей в японские йены. Выберите из RUB, в JPY, введите 5000 и нажмите кнопку. Результат составит примерно 8200 JPY.

Проверка валидации: попробуйте ввести 0 или отрицательное число, например -50. Приложение выдаст ошибку "Сумма должна быть положительным числом". Введите 150.75 — конвертация выполнится корректно.

Сохранение истории: после каждой успешной конвертации запись автоматически добавляется в таблицу истории и сохраняется в файл history.json. Вы можете закрыть и снова открыть приложение — история загрузится автоматически. Кнопка "Очистить историю" удаляет все записи как из таблицы, так и из файла. Кнопка "Обновить курс" показывает текущий курс между выбранными валютами без выполнения конвертации.
